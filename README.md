## 功能
- [x] 自定义文字
- [x] 黑底白字
- [x] 白底黑字
- [x] 黑底红字
- [x] 模糊
- [x] 噪点
- [x] 锐化
- [x] 模仿95版画面
- [x] 文字转繁体
- [x] 提示当前缺失字型 `2022-07-08`
- [x] 一键尝试替换缺失字型 `2022-07-08`
- [x] 输出画面比例 `2022-09-03`

## 说明
 - 本机安装 **Matisse-EB** 字体的情况会得到最佳生成体验
 - 实在找不到替换文字时可以偷懒安装一份 **思源宋体 Heavy** 字型作为 `Fallbacks`
 - 字体授权来自 **fontwork**，仅供个人使用。再利用需自行购买授权
